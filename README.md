# ⏱ iOSApp3 – Timer App

## 📱 Overview
iOSApp3 is a simple SwiftUI timer app that lets users start countdown timers using preset durations. It includes a circular progress ring to visually show timer progress.

---

## ✨ Features
- Preset timers (1, 5, 10, 20, 30 minutes)
- Circular progress indicator
- Smooth animations
- Simple navigation between screens
- MVVM architecture

---

## 🧱 Project Structure

iOSApp3
│
├── Views
│ ├── PresetTimerView.swift
│ ├── TimerView.swift
│ └── ProgressRing.swift
│
├── ViewModels
│ └── TimerViewModel.swift
│
└── IOSApp3App.swift


---

## ⚙️ How It Works
- User selects a preset timer
- App opens TimerView
- Timer starts countdown
- ProgressRing shows progress visually
- Timer completes when it reaches zero
